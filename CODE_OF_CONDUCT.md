# Code of conduct

## Our commitment

We are committed to a welcoming, safe, and harassment-free community for everyone, regardless of background, identity, experience, or level of technical expertise.

## Expected behavior

- Be respectful, specific, and constructive.
- Critique ideas and code, not people.
- Assume good intent while taking security and privacy concerns seriously.
- Respect confidentiality, especially for vulnerability reports and forensic data.
- Accept responsibility, apologize, and repair harm when mistakes happen.

## Unacceptable behavior

Harassment, discrimination, threats, sexualized attention, sustained disruption, doxxing, publishing another person's private data, or retaliating against good-faith security research is not tolerated.

## Enforcement

Report conduct concerns privately through the repository owner's GitHub profile or repository moderation tools. Maintainers may remove content, restrict participation, or ban contributors when necessary. Reports will be handled as confidentially as reasonably possible.

This code applies in project spaces and whenever someone publicly represents the project.
